Our Innovation

Intelligent Adaptive Control

We replace fixed scheduling with a closed-loop system that continuously sees, calculates, schedules, and validates traffic decisions.

1. See (Perception)

YOLOv8 monitors camera feeds, counting cars, buses, and ambulances.

2. Think (Optimizer)

Priority Scheduler processes weights and assigns optimal green times.

3. Route (API Hub)

Secured API coordinates commands and pushes WebSocket state packages.

4. Act (SUMO / Grid)

Dynamic traffic signal controls are simulated and deployed.

Under The Hood

System Pipeline & Data Flow

1

Perception Engine

Performs object detection using a fine-tuned YOLOv8 Nano model on OpenCV video frame streams at 10 FPS.

Python 3.11 YOLOv8 OpenCV ZeroMQ PUSH
2

Optimizer Brain

Applies mathematical weights to vehicle classes and schedules phases using a negated Max-Heap Priority Queue. Allocations are clamped between 10s-90s.

Python 3.11 Negated Min-Heap ZeroMQ PULL/PUB
3

Command API Server

Express backend receives optimizer updates via ZeroMQ, runs a 1-second countdown ticker, and broadcasts WebSocket states. Exposes JWT-secured REST routes for overrides.

Node.js Socket.io JWT auth Express
4

Simulation Validation

Subscribes to the optimizer state feed. Uses TraCI API to force red/yellow/green states on a simulated sumo traffic junction network in real-time.

SUMO GUI TraCI API ZeroMQ SUB
Technology Stack

Architectural Specifications

AI & Vision

  • YOLOv8 Nano Model: Fast, high-accuracy object detection (< 15ms latency).
  • OpenCV: Handles frame decodings, ROIs, and dashboard overlays.

Messaging & IPC

  • ZeroMQ (pyzmq / zeromq.js): High-throughput, low-latency TCP communication.
  • Socket.io: Handles push updates to active operator web interfaces.

Web & Security

  • Express.js Router: Standard REST API.
  • JWT HS256 Auth: Encrypted tokens for secure traffic operator overrides.
  • Helmet & Rate-Limits: Middleware hardening.

Simulation & Logs

  • SUMO (Eclipse): Microscopic traffic simulation.
  • Pandas & SQLite: Persistent storage for traffic metrics logging.
